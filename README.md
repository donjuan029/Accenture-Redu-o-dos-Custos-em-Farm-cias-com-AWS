# Implementação de Serviços AWS para Redução de Custos em Farmácias

Este repositório contém um **relatório de implementação de serviços da Amazon Web Services (AWS)** com foco na **redução de custos operacionais em empresas do setor farmacêutico**.

O projeto apresenta uma proposta de adoção de tecnologias em nuvem que permitem substituir infraestrutura física por serviços escaláveis e sob demanda.

---

# 📄 Sobre o Projeto

O objetivo deste projeto é demonstrar como a utilização de serviços da **AWS** pode ajudar farmácias e empresas do setor de saúde a:

* reduzir custos com infraestrutura de TI
* melhorar a escalabilidade dos sistemas
* aumentar a disponibilidade dos serviços
* automatizar processos operacionais
* armazenar dados de forma segura

O relatório apresenta **três serviços da AWS** que podem ser implementados rapidamente e gerar benefícios imediatos.

---

# ☁️ Serviços AWS Utilizados

## Amazon EC2

Serviço de **computação em nuvem** que permite executar servidores virtuais sob demanda.

**Benefícios:**

* elimina necessidade de servidores físicos
* escalabilidade conforme demanda
* pagamento apenas pelos recursos utilizados

**Exemplo de uso em farmácias:**

* sistema de gestão de estoque
* sistema de vendas
* aplicações internas da empresa

---

## Amazon S3

Serviço de **armazenamento de objetos altamente seguro e escalável**.

**Benefícios:**

* armazenamento de documentos e backups
* alta durabilidade dos dados
* baixo custo de armazenamento

**Exemplo de uso em farmácias:**

* notas fiscais digitais
* relatórios financeiros
* backups de sistemas
* documentos administrativos

---

## AWS Lambda

Serviço **serverless** para execução de código sem necessidade de gerenciar servidores.

**Benefícios:**

* pagamento apenas pelo tempo de execução
* automação de processos
* redução de custos de infraestrutura

**Exemplo de uso em farmácias:**

* geração automática de relatórios
* processamento de pedidos
* atualização automática de estoque
* integração entre sistemas
